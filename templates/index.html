Selenium

Web otomasyonu için güçlü ve esnek çözüm

Dökümanlar İndir
Test Kodu Örneği
from selenium import webdriver
from selenium.webdriver.common.by import By
import unittest

class TestScenario(unittest.TestCase):
    
    def setUp(self):
        self.driver = webdriver.Chrome()
        self.driver.get("https://example.com")
    
    def test_example(self):
        # Test steps will go here
        pass
        
    def tearDown(self):
        self.driver.quit()
Cypress Entegrasyonu

Cypress

Modern web için JavaScript tabanlı test aracı

Dökümanlar İndir
Test Kodu Örneği
describe('Test Scenario', () => {
  beforeEach(() => {
    cy.visit('https://example.com')
  })
  
  it('should perform the test scenario', () => {
    // Test steps will go here
    cy.get('#element').click()
    cy.contains('Success').should('be.visible')
  })
})
Playwright Entegrasyonu

Playwright

Microsoft'un modern web testleri için geliştirdiği araç

Dökümanlar İndir
Test Kodu Örneği
const { test, expect } = require('@playwright/test');

test('test scenario', async ({ page }) => {
  await page.goto('https://example.com');
  
  // Test steps
  await page.click('#element');
  await expect(page.locator('.success')).toBeVisible();
});
